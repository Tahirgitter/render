To run locally:
1. Make sure Python 3.8+ is installed.
2. (Optional) Create and activate a virtualenv:
   python -m venv venv
   source venv/bin/activate   # on Windows: venv\Scripts\activate
3. Install dependencies:
   pip install -r requirements.txt
4. Run:
   python app.py
5. Open http://127.0.0.1:5000 in your browser.

Notes:
- The app saves progress to your browser localStorage and to the Flask session.
- Two photos were copied from the upload container; the rest are placeholders you can replace in static/images/.
